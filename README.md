## Instalação

Use [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/) para instalar todas as dependencias do projeto.

```bash
npm install
```
ou
```bash
yarn install
```

## Iniciar Projeto

```bash
yarn start
```

## Requisitos
- Node - v14.16.0
